# Security Policy

## Supported Versions

This project only supports the latest release. Please upgrade to the most recent version before reporting security issues.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Preferred contact:
- GitHub Security Advisories: https://github.com/SamXop123/Paraline/security/advisories

If the advisory form is unavailable, open a GitHub issue and mark it as a security concern with minimal detail. A maintainer will follow up with a private channel for full disclosure.

What to include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment
- A proof of concept, if possible

We will acknowledge receipt and work on a fix as quickly as possible.
