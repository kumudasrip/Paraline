# Paraline 1.2.0

Paraline 1.2.0 expands the landing site, refreshes visuals, and polishes packaging plus workflows for a smoother release experience.

## Highlights

- New and improved visuals: Reactive Border enhancements, Snow Particles, and Ripple Flow updates
- Landing site expansion with FAQ, Privacy Policy, Terms, and Installation Guide pages
- Refined UI on the landing site: footer, sidebar, icons, and scrollbar styling
- Packaging polish with a multi-resolution Windows tray/installer icon
- CI and release workflow improvements plus new community/docs files

## What's New

### Visual Themes

- Reactive Border enhancements for richer responsiveness
- Snow Particles theme improvements
- Ripple Flow refinements for smoother motion

### Landing Site

- Added FAQ, Privacy Policy, Terms, and Installation Guide pages
- Added a footer with contact and legal links
- Sidebar improvements and new icon set
- New sitemap and robots.txt

### Packaging and Project

- Updated Windows tray/installer icon assets
- Release and CI workflow updates
- New Code of Conduct and CONTRIBUTING guide

## Notes

- Installer artifact name: `Paraline-Setup-1.2.0.exe`
- No migration required; existing settings remain compatible

## Thank You

Thanks to everyone contributing and testing Paraline. This update keeps the project moving forward with better visuals and a stronger public presence.
